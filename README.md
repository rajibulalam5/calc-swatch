# calc-swatch

---

# ⚡ Utility Interface Project

🧮 Calculator + ⏱ Stopwatch in one clean web app

A fast, minimal dual-tool web interface built using **HTML, CSS, and JavaScript**. Designed for instant calculations and precise time tracking with a smooth, distraction-free experience.

---

## 🚀 What This Is

A combined **calculator + stopwatch tool** that runs directly in the browser.

Built to feel:

* ⚡ Fast
* 🧊 Clean
* 🎯 Focused
* 💡 Instant

---

## ✨ Features

### 🧮 Calculator

* Basic arithmetic operations (+ − × ÷)
* Percentage support
* Square root function
* Toggle sign (+/−)
* Live expression display
* Full keyboard support

### ⏱ Stopwatch

* Start / Pause / Reset controls
* Millisecond precision timing
* Lap recording system
* Clean digital display
* Real-time updates

---

## 🎨 UI Style

* 🌑 Dark minimal interface
* 🔲 Soft bordered card layout
* ⌨️ Monospace numeric display
* 🎯 Zero clutter design philosophy
* ⚡ Instant visual feedback

Everything visible has a purpose.

## 🧠 How It Works

### Calculator Engine

A lightweight JavaScript expression evaluator:

```js id="y9kq3m"
Function('"use strict"; return (' + expr + ')')();
```

✔ Simple
✔ Fast
✔ Direct browser execution

---

### Stopwatch Engine

* 10ms interval timing loop
* Internal time state tracking
* Lap delta calculations
* Real-time UI rendering
